    避免断定的说法：{
        ～のではないか/～のではないだろうか/～のではないでしょうか
        ～ではないか/～ではないだろうか/ではないでしょうか
        
        ～気がする              きがする④                   感觉，好像
        
        ～と思う/～と思える/～と思われる
    }
    陈述预想： {
        ～はずです，基于自我推论，认为理所当然会成立的事情
        ～ようだ/～らしい，基于某种根据作出的判断
    }
    ～のところ，…阶段
    ～でも～でも/～にしても、～にしても，举例
    （あっ，）そうそう、想起、想到，意思同そう言えば
    ～わけではない，部分否定，并非
    ～夢中になって～する，被什么东西吸引而投入其中，还有“拼命的，努力的意思”
    ～という/ということです/～そうだ、表示传闻
    ～向け/～向き、适当的，恰当的、针对某目标
    ～に関(かん)する、相当于～について/に関連（かんれん）して，但更郑重
    ～にしか、表示限定　大人にしか～ない、表示仅限成年人，除成年人外其他都不
    ～だけではなく/だけでなく～も，不仅～，～也... {
        ～のみならず～も
        ～ばかりでなく/ばかりではなく
    }
    ～とは比べ物にならない、比表示“相差很大”
    ～とともに、～，表示“with～，～；随着～，～”，还表示A和B同时进行
    いきいきとしている，表示动植物等生机勃勃的样子
    ～力，表示能做某事的能力
